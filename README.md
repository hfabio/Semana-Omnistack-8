# Semana-Omnistack-8
Stack da semana omnistack 8

Para executar o backend (que estará rodando na porta 3333):
```
    cd backend
    yarn start
```

Para executar o frontend (que estará rodando na porta default, 3000):
```
    cd front-end
    yarn start
```

Para rodar o mobile ():
```
    cd mobile
    yarn start
```
